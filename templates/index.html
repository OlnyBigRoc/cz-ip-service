API 文档

请求参数

参数名 类型 必填 描述
ip string 否 要查询的IP地址，IPv4或IPv6格式。若不提供则使用请求者IP

响应字段

字段名 类型 描述
code int 状态码，200表示成功
data object IP信息数据对象
data.ip
string 查询的IP地址
data.country
string IP归属国家
data.province
string IP归属省份
data.city
string IP归属城市
data.isp
string IP归属运营商
data.time
int64 查询耗时(微秒)
message string 错误信息，仅在code不为200时出现

示例响应

{

    "data": {
        "ip": "8.8.8.8",
        "country": "美国",
        "province": "",
        "city": "",
        "isp": "Google公司",
        "time": 156
    }
}
